# Jogo da Memória

Este projeto foi desenvolvido em aula do curso [B7WEB](https://b7web.com.br).

## Available Scripts

In the project directory, you can run:

### `npm install`

Esse comando irá instalar todas as dependências necessarias para rodar o projeto.

### `npm run dev`

Esse comando vai abrir o projeto em modo de desenvolvimento no seu navegador
